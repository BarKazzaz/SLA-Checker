Documentation

  • Monitor 1st Tier:

    Clicking on "Monitor 1st tier queue" will create a new pinned tab which will be monitored in the background.
    Once a ticket in that queue has 90 minutes or less according to SLA you will be notified.

  • Monitor This Queue

    Clicking on "Monitor this queue" will duplicate current tab to a new pinned tab which will be monitored in the background.
    Once a ticket in that queue has 90 minutes or less according to SLA you will be notified.
    Pleaes make sure you are viewing a queue in order to use this functionality.

  • Stop Monitoring This Queue

    Clicking on "Stop monitoring this queue" will disable monitoring for the specific tab you are on.
    There is no "Enable" button, so if you regret stopping you may use "Monitor this queue" and have a duplicate created.

  • Monitor Priority

    "Monitor Priority" is a toggle button.
    When active (green background) you will get notified when there are tickets in priority queue.
    You have to monitor a queue to use this addon.

  • Mute

    "Mute" is a toggle button.
    When active notifications will have a sound effect.
    When muted notifications will be silent.
